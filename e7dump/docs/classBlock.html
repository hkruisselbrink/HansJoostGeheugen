Detailed Description

Represents a single datablock from the filesystem device.

It contains DBLKSIZ (=512) bytes of raw binairy data.

Examples:
1.voorbeeld.cc, and 2.charter.cc.

ulong Block::usage_counter
protected

Block usage counter.

Keeps track of how often this block is being used at the same time. Incremented by Device.getBlock() when you ask for a Block. Decremented by calling Block.release(). Used by Device.getBlock() for a block replacement algoritm when we limit the size of the block cache.

Examples:
1.voorbeeld.cc.
